# EdgeCraft Quant Lab 0.1

Första versionen av EdgeCraft Quant Lab.

## Innehåll
- Double Seven-strategi
- Val av marknad
- Val av timeframe
- Trendfilter MA
- Entry-lookback
- Exit-lookback
- Resultat: PF, winrate, drawdown, trades, equity curve

## Kör lokalt
pip install -r requirements.txt
streamlit run app.py
